# Rizzume

AI-powered dating profile analyzer that reviews photos, prompts, and bios to provide actionable feedback and personalized conversation starters.

## Features

- Analyze profile photos
- Review bios and prompts
- Generate profile scores
- Suggest profile improvements
- Create personalized opener messages

## Example

Input:
- 6 profile photos
- Dating bio
- Prompts

Output:
- Photo feedback
- Bio feedback
- Overall profile score
- Custom opener suggestions

## Tech Stack

- Python
- FastAPI
- OpenAI API
- React
- PostgreSQL
